## Branch Strategy

| Branch | Purpose | Merge from | Merge type |
|---|---|---|---|
| `main` | Production | `release-*` only | Merge commit |
| `develop` | Integration | Feature branches | Squash merge |
| `feature/*` | New work | `develop` | — |
| `release/*` | Release prep | `develop` | — |
| `fix/*` | Bug fixes | `develop` | — |

## Enforced Rules

- Linear git history — no merge commits on feature branches (squash or rebase only)
- No direct push to `main` or `develop` — PRs required
- No force push on protected branches

## Common Commands

### Daily workflow
```bash
# Start new work
git checkout develop
git pull origin develop
git checkout -b feature/my-feature

# Save work in progress
git stash
git stash pop

# Keep branch up to date
git fetch origin
git rebase origin/develop
```

### Committing
```bash
# Stage and commit
git add .
git commit -m "type: short description"

# Amend last commit (before push only)
git commit --amend --no-edit
```

### Pushing and PRs
```bash
# Push feature branch
git push -u origin feature/my-feature

# After PR is merged — clean up
git checkout develop
git pull origin develop
git branch -d feature/my-feature
```

## Recovery Guide

### Restore a deleted branch
```bash
# Find the commit hash the branch pointed to
git reflog

# Recreate the branch
git checkout -b feature/my-feature <commit-hash>
git push -u origin feature/my-feature
```

### Undo last commit (keep changes)
```bash
git reset --soft HEAD~1
```

### Undo last commit (discard changes)
```bash
git reset --hard HEAD~1
# WARNING: changes are lost permanently
```

### Recover a lost stash
```bash
git fsck --no-reflog | grep commit | awk '{print $3}' | xargs git log --oneline --no-walk
# Find your stash commit, then:
git stash apply <commit-hash>
```

### Resolve merge conflicts
```bash
# After a conflict during rebase or merge:
# 1. Open conflicted files, resolve manually
# 2. Stage resolved files
git add <resolved-file>
# 3. Continue
git rebase --continue   # or: git merge --continue
```

### Undo a pushed commit (without rewriting history)
```bash
git revert <commit-hash>
git push origin <branch>
```
